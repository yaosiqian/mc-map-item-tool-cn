我应该如何处理地图文件？

  1. 生成一个地图
  2. 打开世界文件夹
  3. 将你生成并下载的 map_[地图编号].dat 移动到 data 文件夹
  4. 重新载入世界

地图文件是如何生成的？

该项目使用 node-nbt 来从一个 JavaScript 对象创建一个NBT格式的文件

该项目用了什么黑科技？

HTML5 文件API，网页通知，Canvas，JavaScript (jQuery 和 bootstrap 用于网页布局，Node.js 用于服务器，Colour.js 用于调整 Minecraft 的颜色)

为什么这个工具会问我“之前生成了多少地图”？

这个编号用来给生成的地图文件提供正确的文件名。例如，如果您想在您的世界中替换第 114514 号地图，您只需输入 "114514"。当使用分割图像功能时，生成的压缩文件中的所有地图文件将根据您的输入获得它们的编号。如果您输入 "15"，并将图像分割成4个部分，文件将有15、16、17和18的数字。
